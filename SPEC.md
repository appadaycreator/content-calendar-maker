# コンテンツカレンダー作成ツール【無料】自動作成ツール - 技術仕様書

## 概要

**サービス名**: Content Calendar Maker
**バージョン**: 1.2.0
**更新日**: 2026-06-10
**URL**: https://appadaycreator.com/content-calendar-maker/

ブログ・SNS・YouTube向けのコンテンツ投稿カレンダーを自動生成。登録不要・完全無料でご利用いただけます。

## 新機能（v1.2.0）

### M3: データ保存・履歴機能（localStorage）
- **自動保存**: カレンダー生成時に入力値・結果をlocalStorageに自動保存
- **前回データ復元**: 「前回データを復元」ボタンで直前の設定を1クリック復元
- **履歴表示**: 前回保存の日時表示で確認可能

### M4: グラフ・可視化（Chart.js）
- **投稿数統計**: 投稿総数・平均投稿数/週を表示
- **プラットフォーム別分析**: ドーナツチャートで各チャネルの投稿比率を可視化
- **ジャンル別分析**: 横棒グラフでジャンル別の投稿数をランク表示

### 既存機能

#### v1.1.0: 表示形式の切り替え
- **グリッド表示**：従来のカレンダー形式
- **リスト表示**：投稿を日付順の縦列で表示（モバイルに最適）
- PC・モバイル両対応の表示切り替えボタン

#### v1.1.0: マルチフォーマットエクスポート
- **CSV**：従来どおり、Excel・スプレッドシート対応
- **JSON**：API連携・プログラム処理向け
- **Excel（TSV形式）**：タブ区切りでExcelで直接開可能

## データ管理

- **ストレージ**: ブラウザ localStorage（外部API通信なし）
- **永続化**: ページ読み込み時に自動復元
- **クリア**: ブラウザのサイトデータ削除で初期化

## 技術スタック

- HTML5 / CSS3 / Vanilla JavaScript
- PWA対応（manifest.json / Service Worker）
- レスポンシブデザイン（モバイルファースト）

## 使い方

1. ページを開き、入力フォームの項目を確認する
2. 必要な情報を入力または選択する
3. 実行ボタンをクリックして結果を取得する
4. 表示された結果・アドバイスを確認する
5. 必要に応じてコピー・SNSシェアで活用する

## よくある質問（FAQ）

**Q: コンテンツカレンダー作成ツールは無料で使えますか？**

はい、完全無料・登録不要でご利用いただけます。

**Q: 何回でも使えますか？**

はい、回数制限なく何度でもご利用いただけます。

**Q: 入力したデータはサーバーに送信されますか？**

いいえ。すべての処理はブラウザ内で完結し、入力内容はサーバーへ送信されません。

**Q: スマートフォンでも使えますか？**

はい、スマートフォン・タブレット・PCすべてに最適化されています。

**Q: 結果を保存・共有できますか？**

スクリーンショットでの保存またはSNSシェアボタンからご共有いただけます。
